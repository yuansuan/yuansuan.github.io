    文件管理

    可在「文件管理」中组织、管理模型文件、作业结果等文件，支持上传文件/文件夹、新建文件夹、下载、移动、删除等操作。

     
    平台右上侧显示已用空间容量和空间总容量，当已用容量超过总容量时，云存储会弹性扩展，但强烈建议用户进行文件清理，避免产生额外费用。
